Russian Roulette

Instructions

Objective:
Survive the game by keeping your lives while trying to deplete the lives of your opponent.

Setup:
Player 1 and Player 2 start with an equal number of lives (♥♥♥).
A revolver with entered number of blanks and live bullets is used.
The game initializes by spinning the chamber and randomly placing the bullets.

Start the Game:
Enter the number of blanks and live bullets when prompted.
Click the button to initialize the game.

Player's Turn:
The game indicates whose turn it is by displaying Player 1's Turn or Player 2's Turn.
Players have two main actions: and

Shoot Opponent:
Click button to attempt shooting the opponent.
If a blank bullet is fired, the turn passes to the next player.
If a live bullet is fired, the opponent loses a life.
If the opponent is wearing a vest, the vest gets destroyed instead of losing a life. Each player has one vest.

Shoot Yourself:
Click to attempt shooting yourself.
If a blank bullet is fired, you survive, and you get an extra turn.
If a live bullet is fired, you lose a life.
If you are wearing a vest, the vest gets destroyed instead of losing a life.

Special Actions:
(Can only be used once!)
: Each player can use the Check Next Bullet action once per game to check if the next bullet is a blank or live. This does not change the order of the bullets.
: Each player starts with a vest that can absorb one live bullet. Once used, it is destroyed. It can shield you for the opponent's next 2 turns.

Winning the Game:
The game continues until one player loses all their lives.
The player who still has lives left is declared the winner.

Notes:
The game involves random chance, strategy, and a bit of luck.
Use the vest wisely as it can be the difference between winning and losing.
Enjoy the game and may the odds be ever in your favor!
